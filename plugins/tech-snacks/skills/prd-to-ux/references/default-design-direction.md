# Default Design Direction (shadcn-style neutral hi-fi)

Loaded inline by Step 4 when the user skips Step 3. Never written to disk.

## Mood & Personality
Neutral, modern, trustworthy. Gets out of the user's way. Confident without shouting.

## Tone
Quietly utilitarian. Clarity over flourish.

## Palette
Black, white, a measured range of grays. One understated accent used sparingly for primary actions.

## Type
Modern sans-serif. Clear hierarchy between heading and body weights. Generous line-height on body text.

## Layout
Generous whitespace. Comfortable margins. Content-first — chrome recedes.

## Surfaces
Subtle shadows, thin borders, restrained corner radii. No decorative gradients or ornament.

## Components
Clean, quiet form fields. Primary action is visually distinct but not loud. Secondary actions are obvious but never compete.

## Interactions
Calm transitions. No bounce, no springy flourish. State changes are legible, not performative.
